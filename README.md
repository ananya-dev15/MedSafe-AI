# MedSafe AI

MedSafe AI is an AI-powered healthcare assistance platform designed to improve medicine safety awareness and symptom understanding.

## Features

- Medicine Interaction Checker
- Prescription OCR Reader
- Symptom Guidance System
- Side Effect Monitoring
- Emergency Risk Predictor

## Technologies Used

- Python
- Streamlit
- EasyOCR
- RapidFuzz
- Pandas

## How to Run

Install dependencies:

pip install -r requirements.txt

Run the application:

streamlit run app.py
